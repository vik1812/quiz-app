# Quiz Microeconomia (HTML/CSS/JS)

Quiz locale per studiare Microeconomia. Funziona su GitHub Pages.

## Struttura
- index.html
- style.css
- script.js
- questions.json

## Aggiungere domande
Apri `questions.json` e aggiungi elementi con:
- id (string)
- topic (string, opzionale)
- question (string)
- choices (array di stringhe)
- answerIndex (numero, 0-based)
- explanation (string, opzionale)
